# Rank Checker API

> **Status**: DEPRECATED
>
> **Deprecation Notice**: This API has been deprecated and is no longer supported.
>
> **Replacement**: [Local Rank Tracker API](https://brightlocal.stoplight.io/docs/management-apis/zmd64hm2pgta3-local-rank-tracker)

## Overview

The Rank Checker API was previously used to track local search rankings for businesses across various search engines. This functionality has been migrated to the new Local Rank Tracker API.

## Migration Guide

All users of the Rank Checker API should migrate to the new Local Rank Tracker API, which provides:

- Enhanced ranking tracking capabilities
- Improved accuracy and reporting
- Better integration with BrightLocal's management platform

### Key Differences

| Feature | Old Rank Checker API | New Local Rank Tracker API |
|---------|---------------------|---------------------------|
| Base URL | tools.brightlocal.com/seo-tools/api | developer.brightlocal.com |
| Documentation | apidocs.brightlocal.com | brightlocal.stoplight.io |
| Authentication | API key parameter | API key (updated method) |

## Legacy Endpoints (No Longer Functional)

The following endpoints were part of the Rank Checker API but are no longer available:

- Ranking report creation
- Ranking report management
- Ranking results retrieval

## Support

For assistance migrating to the new Local Rank Tracker API, contact BrightLocal support or refer to the [new API documentation](https://brightlocal.stoplight.io/docs/management-apis/zmd64hm2pgta3-local-rank-tracker).
