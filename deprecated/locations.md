# Locations API

> **Status**: DEPRECATED
>
> **Deprecation Notice**: This API has been deprecated and is no longer supported.
>
> **Replacement**: [Locations API (New)](https://developer.brightlocal.com/docs/management-apis/a8chirfprjb2p-locations)

## Overview

The Locations API was previously used to manage location/business records within the BrightLocal platform, including creating, updating, and organizing location data for multi-location businesses and agencies.

## Migration Guide

All users of the old Locations API should migrate to the new Locations API, which provides:

- Updated endpoint structure
- Enhanced location management capabilities
- Improved data organization
- Better integration with BrightLocal's management platform

### Key Differences

| Feature | Old Locations API | New Locations API |
|---------|------------------|-------------------|
| Base URL | tools.brightlocal.com/seo-tools/api | developer.brightlocal.com |
| Documentation | apidocs.brightlocal.com | developer.brightlocal.com |
| Authentication | API key parameter | API key (updated method) |

## Legacy Functionality

The old Locations API included functionality for:

- Creating location records
- Updating location information (name, address, phone, website, etc.)
- Listing and searching locations
- Deleting location records
- Managing location metadata and associations

All of this functionality is available in the new API with improved features.

## Support

For assistance migrating to the new Locations API, contact BrightLocal support or refer to the [new API documentation](https://developer.brightlocal.com/docs/management-apis/a8chirfprjb2p-locations).
