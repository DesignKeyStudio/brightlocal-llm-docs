# Rankings API

> **Status**: DEPRECATED
>
> **Deprecation Notice**: This API has been deprecated and is no longer supported.
>
> **Replacement**: [Rankings API (New)](https://developer.brightlocal.com/docs/data-apis/a65f86e53ce22-rankings-api)

## Overview

The Rankings API was previously used to track and monitor search engine rankings for businesses across various keywords and locations. This functionality has been migrated to the new Rankings API on the developer portal.

## Migration Guide

All users of the old Rankings API should migrate to the new Rankings API, which provides:

- Enhanced ranking tracking capabilities
- Improved data accuracy
- Better reporting and analytics
- Updated platform integration

### Key Differences

| Feature | Old Rankings API | New Rankings API |
|---------|-----------------|------------------|
| Base URL | tools.brightlocal.com/seo-tools/api | developer.brightlocal.com |
| Documentation | apidocs.brightlocal.com | developer.brightlocal.com |
| Authentication | API key parameter | API key (updated method) |

## Legacy Functionality

The old Rankings API included functionality for:

- Creating ranking reports
- Tracking keyword positions
- Monitoring competitor rankings
- Generating ranking trend data

All of this functionality is available in the new API with improved features.

## Support

For assistance migrating to the new Rankings API, contact BrightLocal support or refer to the [new API documentation](https://developer.brightlocal.com/docs/data-apis/a65f86e53ce22-rankings-api).
