# Citation Builder API

> **Status**: DEPRECATED
>
> **Deprecation Notice**: This API has been deprecated and is no longer supported.
>
> **Replacement**: [Citation Builder API (New)](https://developer.brightlocal.com/docs/management-apis/agjq3z0pfs6nk-citation-builder)

## Overview

The Citation Builder API was previously used to manage citation building campaigns for businesses, including submitting business information to local directories and tracking citation progress.

## Migration Guide

All users of the old Citation Builder API should migrate to the new Citation Builder API, which provides:

- Updated endpoint structure
- Improved campaign management
- Enhanced reporting capabilities
- Better integration with the BrightLocal platform

### Key Differences

| Feature | Old Citation Builder API | New Citation Builder API |
|---------|-------------------------|-------------------------|
| Base URL | tools.brightlocal.com/seo-tools/api | developer.brightlocal.com |
| Documentation | apidocs.brightlocal.com | developer.brightlocal.com |
| Authentication | API key parameter | API key (updated method) |

## Legacy Functionality

The old Citation Builder API included functionality for:

- Creating citation building campaigns
- Managing directory submissions
- Tracking citation progress
- Retrieving campaign results

All of this functionality is available in the new API with improved features.

## Support

For assistance migrating to the new Citation Builder API, contact BrightLocal support or refer to the [new API documentation](https://developer.brightlocal.com/docs/management-apis/agjq3z0pfs6nk-citation-builder).
