# Clients API

> **Status**: DEPRECATED
>
> **Deprecation Notice**: This API has been deprecated and is no longer supported.
>
> **Replacement**: [Clients API (New)](https://developer.brightlocal.com/docs/management-apis/j13dhiwfwfldc-clients)

## Overview

The Clients API was previously used to manage client records within the BrightLocal platform, including creating, updating, and organizing client accounts for agencies and multi-location businesses.

## Migration Guide

All users of the old Clients API should migrate to the new Clients API, which provides:

- Updated endpoint structure
- Enhanced client management capabilities
- Improved data organization
- Better integration with BrightLocal's management platform

### Key Differences

| Feature | Old Clients API | New Clients API |
|---------|----------------|-----------------|
| Base URL | tools.brightlocal.com/seo-tools/api | developer.brightlocal.com |
| Documentation | apidocs.brightlocal.com | developer.brightlocal.com |
| Authentication | API key parameter | API key (updated method) |

## Legacy Functionality

The old Clients API included functionality for:

- Creating client records
- Updating client information
- Listing and searching clients
- Deleting client records
- Managing client metadata

All of this functionality is available in the new API with improved features.

## Support

For assistance migrating to the new Clients API, contact BrightLocal support or refer to the [new API documentation](https://developer.brightlocal.com/docs/management-apis/j13dhiwfwfldc-clients).
