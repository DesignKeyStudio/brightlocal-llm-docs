# Reputation Manager API

> **Status**: DEPRECATED
>
> **Deprecation Notice**: This API has been deprecated and is no longer supported.
>
> **Replacement**: [Reputation Manager API (New)](https://developer.brightlocal.com/docs/management-apis/p7axgeo0csqf5-reputation-manager)

## Overview

The Reputation Manager API was previously used to manage online reputation monitoring, including tracking reviews, managing review responses, and generating reputation reports.

## Migration Guide

All users of the old Reputation Manager API should migrate to the new Reputation Manager API, which provides:

- Enhanced review monitoring capabilities
- Improved response management
- Better analytics and reporting
- Updated platform integration

### Key Differences

| Feature | Old Reputation Manager API | New Reputation Manager API |
|---------|---------------------------|---------------------------|
| Base URL | tools.brightlocal.com/seo-tools/api | developer.brightlocal.com |
| Documentation | apidocs.brightlocal.com | developer.brightlocal.com |
| Authentication | API key parameter | API key (updated method) |

## Legacy Functionality

The old Reputation Manager API included functionality for:

- Setting up reputation monitoring campaigns
- Tracking reviews across multiple platforms
- Managing review responses
- Generating reputation reports
- Configuring alerts and notifications

All of this functionality is available in the new API with improved features.

## Support

For assistance migrating to the new Reputation Manager API, contact BrightLocal support or refer to the [new API documentation](https://developer.brightlocal.com/docs/management-apis/p7axgeo0csqf5-reputation-manager).
